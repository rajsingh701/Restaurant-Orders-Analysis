# Restaurant Orders Analysis

## 📌 Project Overview
This project analyzes restaurant sales and customer order data using **Python, Pandas, and Jupyter Notebook**. The objective is to uncover ordering patterns, identify top-performing menu items, evaluate revenue generation, and provide actionable business insights that can help improve restaurant operations and profitability.

By combining menu and order datasets, this analysis explores customer behavior, sales trends, and product performance through data cleaning, transformation, and exploratory data analysis (EDA).

---

## 🎯 Objectives

- Analyze restaurant sales performance.
- Identify the most and least popular menu items.
- Calculate total revenue and average order value.
- Discover peak ordering periods.
- Evaluate category-wise revenue contribution.
- Understand customer ordering behavior.
- Generate insights for menu optimization and business growth.
